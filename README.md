# Fractal Drawing

Этот проект — простая HTML-страница, рисующая фрактал с помощью элемента `<canvas>` и JavaScript.

## Как запустить

1. Сохраните код в файл, например `fractal.html`.
2. Откройте его в любом современном браузере.

## Как работает

* Используется рекурсивная функция, которая генерирует фракталы.
* При достижении глубины `depth = 0` рисуются маленькие красные квадраты (`fillRect`).
* Изменяя параметр `depth`, можно контролировать уровень детализации фрактала.
